# Folder Notebooks
Simpan file Jupyter Notebook (.ipynb) untuk eksplorasi data dan eksperimen model di folder ini.

## Notebook yang Direkomendasikan
1. `01_exploratory_data_analysis.ipynb` — EDA dataset finansial
2. `02_text_preprocessing.ipynb` — Eksperimen tokenization & embedding
3. `03_model_training.ipynb` — Training dan evaluasi model
4. `04_model_comparison.ipynb` — Perbandingan ANN vs LSTM vs GRU

## Cara Menjalankan
```bash
cd data-science
pip install jupyter
jupyter notebook
```
